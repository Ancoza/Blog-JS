Primeros pasos JavaScript

Para iniciar a programar en JavaScript, primero debemos aprender como podemos trabajar en conjunto con un documento HTML, eso significa, que debemos aprender como enlazar un documento JavaScript con un documento HTML.

Podemos trabajar JavaScript en conjunto con HTML de dos maneras. La primera es usando la etiqueta de HTML "script" y de esa manera escribir nuestro codigo en el mismo documento HTML, de la siguiente manera: 

                        
                            <html>
                                <head>
                                    <tittle>Mi primera pagina web</html>
                                </head>
                                <body>
                                    <script>
                                        var i = 5;
                                    </script>
                                </body>
                            </html>

La otra forma de trabajar con JavaScript y HTML en conjunto, es enlazando un documento mediente la etiqueta "script" y su atributo "src", de la siguiente manera:
HTML:

                        
                            <html>
                                <head>
                                    <tittle>Mi primera pagina web</html>
                                </head>
                                <body>
                                    <script src="archivo.js"></script>
                                </body>
                            </html>

JavaScript: 

                        
                            var i = 5;

Variables en JS (var, let y const)

En JS (JavaScript) tenemos distintas formas de declarar variables, al igual que en otros lenguajes, pero en este, en particular, se hace de manera muy facil.
Primero que nada es importante aclarar que tenemos 3 palabras resevadas para declarar variables, las cuales son, var, let y const.

1. var: Esta se usa para declarar una variable global o local, la cual no tiene restricciones de acceso.
2. let: Esta se usa para declarar una variable limitando su alcance, es decir, esta solo usara en un solo ambito.
3. const: La mayor particularidad de esta manera de declarar una variable, es que la palabra reservada const evita que el valor inicial de la variable sea cambiado.

Otro aspecto muy importante a mencionar son los tipos de datos. En JS no es necesario usar las palabras reservadas Int, String, Double, Float para especificar el tipo de dato a usar.

Si por ejemplo quisieramo declarar una variable de tipo String, simplemente tendriamos que escribir la palabra var, let o const (dependiendo de querramos usar), luego el nombre de la variable y se iguala al valor de la variable entre comillas dobles, posteriormente se cierra la linea de codigo con ";" de la siguiente manera: 

                    
                        var nombre = "Emiliano";
                        let nombre = "Marvin";
                        const nombre = "Marco";

Imprimir mensajes en consola JS

Anteriormente vimos el uso correcto y declaración de variables en JS (JavaScript), pero, ahora aprenderemos como mostrar el valor de una variable en la consola del navegador.

Para imprimir en valor de una variable en la consola, debemos usar las propiedad "console.log()". Para usarla, simplemente debemos declararla y dentro de los parentesis escribiremos el nombre la variable cuyo valor queremos mostrar, de la siguiente manera: 

                    
                        var nombre = "Emiliano";
                        
                        console.log(nombre);

Concatenar cadenas en JS

En JS exiten muchas maneras de concatenar cadenas de texto o incluso concatenar una cadena de texto con un valor numerico. Primero aprenderemos como concatenar cadenas de texto.
Para hacerlo, supongamos que tenemos tres variables las cuales almacenan texto: 

                    
                        var nombre = "Emiliano"
                        var apellido = "Urbina"
                        var domicilio = "Ciudad Sandino"

Ahora, imaginemos que queremos mostrar el valor de dichas variables en un solo mensaje. Podriamos unir todas las variables en una variable global que contenga los valores de dichas variables, de la siguiente manera: 

                    
                        var nombre = "Emiliano";
                        var apellido = "Urbina";
                        var domicilio = "Ciudad Sandino";

                        var mensaje = nombre +" "+ apellido +" "+ domicilio;
                        console.log(mensaje);

Otra forma de hacerlo, es concatenando las variables directamente en el "console.log()", de esta manera: 

                    
                        var nombre = "Emiliano";
                        var apellido = "Urbina";
                        var domicilio = "Ciudad Sandino";

                        console.log(nombre +" "+ apellido +" "+ domicilio);

Incluso podriamos escribir directamente los mensajes en el "console.log()" y concatenar con otras variables que contengan texto: 

                    
                        var nombre = "Emiliano";
                        var apellido = "Urbina";
                        var domicilio = "Ciudad Sandino";

                        console.log(nombre +" "+ apellido +" "+"vive en"+" "+ domicilio);

También podriamos usar esta sintaxis y obtendriamos exactamente el mismo resultado: 

                    
                        var nombre = "Emiliano";
                        var apellido = "Urbina";
                        var domicilio = "Ciudad Sandino";

                        console.log(nombre, apellido, domicilio);

Por ultimo, pero no menos importante, podemos usar la sintaxis propuesta por ES6 (EcmaScript 6), la cual nos permite concatenar el valor de una variable (Aunque tenga un valor numerico)
usando estas comillas `` (las obtenemos con la combinación de teclas alt + 96), y dentro de ellas, escribir el mensaje que querramos mostrar en conjunto con nombres de variables cuyos valores querramos mostrar, siempre y cuando dichas variables esten dentro de el siguiente formato de simbolos "${nombre de la variable}", de la siguiente manera: 

                    
                        var nombre = "Emiliano";
                        var apellido = "Urbina";
                        var domicilio = "Ciudad Sandino";
                        var edad = 20;

                        console.log(`${nombre} ${apellido} vive en ${domicilio} y tiene ${edad} años.`);

Operadores lógicos JS

En JS, al igual que en otros lenguajes de programación, podemos hacer operación aritmeticas mediante el uso de operadores como +, -, *, /, para suma, resta, multiplicación y división respectivamente.
Para hacerlo podriamos almacenar el valor de la suma, resta, multiplicación o división dentro de otra variable y sumar los valores de dichas variables como si de dos numeros se tratase, de la siguiente manera: 

                    
                        var num1 = 15; 
                        var num2 = 5; 
                        var suma = num1 + num2; 
                        var resta = num1 - num2;
                        var multiplicación = num1 * num2;
                        var division = num1 / num2;
                        
                        console.log(`Esta es la suma: ${suma}. Esta es la resta: ${resta}. Esta es la multiplicación: ${multiplicación}. Esta es la división: ${division}`)

Otra manera de hacerlo es operando directamente en el lugar donde se desea mostrar el resultado de la operación, de la siguiente manera: 

                    
                        var num1 = 15; 
                        var num2 = 5; 
                        
                        console.log(
                            `Esta es la suma: ${num1 + num2}. Esta es la resta: ${num1 - num2}. 
                            Esta es la multiplicación: ${num1 * num2}. Esta es la división: ${num1 / num2}`
                        )

Por ultimo, podemos reasignar el valor de la variable sumandole el valor de otra variable, es decir, sin necesidad de una tercera variable para contener el resultado de la operación, podriamos realizar la operación dentro de alguna de las dos variables operando, siempre teniendo cuidado del orden en las operaciones de resta y divisón pues no son conmutativas y pueden darnos resultados erroneos si no las ubicamos bien. Lo podemos hacer de la siguiente manera: 

                    
                        var num1 = 15; 
                        var num2 = 5; 
                        
                        num1 = num1 + num2;
                        console.log(num1)

Podemos abreviarlo un poco mas: 

                    
                        var num1 = 15; 
                        var num2 = 5; 
                        
                        num1 += num2;
                        console.log(num1);
