Primeros pasos JavaScript

Para iniciar a programas en JavaScript, primero debemos aprender como podemos trabajar en conjunto con un documento HTML, eso significa, que debemos aprender como enlazar un documento JavaScript con un documento HTML.

Podemos trabajar JavaScript en conjunto con HTML de dos maneras. La primera es usando la etiqueta de HTML "script" y de esa manera escribir nuestro codigo en el mismo documento HTML, de la siguiente manera: 

                        
                            <html>
                                <head>
                                    <tittle>Mi primera pagina web</html>
                                </head>
                                <body>
                                    <script>
                                        var i = 5;
                                    </script>
                                </body>
                            </html>

La otra forma de trabajar con JavaScript y HTML en conjunto es enlazando un documento mediente la etiqueta "script" y su atributo "src", de la siguiente manera:
HTML:

                        
                            <html>
                                <head>
                                    <tittle>Mi primera pagina web</html>
                                </head>
                                <body>
                                    <script src="archivo.js"></script>
                                </body>
                            </html>

JavaScript: 

                        
                            var i = 5;

Variables en JS (var, let y const)

En JS (JavaScript) tenemos distintas formas de declarar variables, al igual que en otros lenguajes, pero en este, en particular, se hace de manera muy facil.
Primero que nada es importante aclarar que tenemos 3 palabras resevadas para declarar variables, las cuales son, var, let y const.

1. var: Esta se usa para declarar una variable global o local, la cual no tiene restricciones de acceso.
2. let: Esta se usa para declarar una variable limitando su alcance, es decir, esta solo usara en un solo ambito.
3. const: La mayor particularidad de esta manera de declarar una variable, es que la palabra reservada const evita que el valor inicial de la variable sea cambiado.

Otro aspecto muy importante a mencionar son los tipos de datos. En JS no es necesario usar las palabras reservadas Int, String, Double, Float para especificar el tipo de dato a usar.

Si por ejemplo quisieramo declarar una variable de tipo String, simplemente tendriamos que escribir la palabra var, let o const (dependiendo de querramos usar), luego el nombre de la variable y se iguala al valor de la variable entre comillas dobles, posteriormente se cierra la linea de codigo con ";" de la siguiente manera: 

                    
                        var nombre = "Emiliano";
                        let nombre = "Marvin";
                        const nombre = "Marco";
